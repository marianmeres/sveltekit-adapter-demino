## Svelte MCP server

You have access to the Svelte MCP server with comprehensive Svelte 5 documentation.
Note: The server also covers SvelteKit, but this project does NOT use SvelteKit — ignore SvelteKit-specific docs.

### Available Tools
1. **list-sections**: Use FIRST to discover available documentation sections.
2. **get-documentation**: Fetch full docs for specific sections.
3. **svelte-autofixer**: Analyze Svelte code for issues — always run before finalizing.
4. **playground-link**: Generate a Svelte Playground link (only after user confirmation).
